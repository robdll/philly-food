# Fonts Directory

Place your custom font files here:

## Required Fonts

1. **Badang Italic**
   - `Badang-Italic.woff2`
   - `Badang-Italic.woff`

2. **Brush Up Too**
   - `Brush-Up-Too.woff2`
   - `Brush-Up-Too.woff`

## Optional Fonts

Work Sans is loaded from Google Fonts by default. If you want to use local files instead:

- `WorkSans-Regular.woff2` / `.woff`
- `WorkSans-Medium.woff2` / `.woff`
- `WorkSans-Bold.woff2` / `.woff`

## Font Sources

If you need to find these fonts:
- Check with your design team
- Search for font licensing websites
- Use similar fonts as fallbacks if unavailable

The site will still work without these custom fonts, but will use system font fallbacks.
